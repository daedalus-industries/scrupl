# scrupl

## Application Development

### Start Blockchain

```
yarn start:blockchain
```

### Migrate Contracts

```
yarn contracts:migrate
```

### Start Trufflepig

```
yarn start:trufflepig
```

### Serve Application

```
yarn start
```

## Contract Development

### Start Blockchain

```
yarn start:blockchain
```

### Compile Contracts

```
yarn contracts:compile
```

### Migrate Contracts

```
yarn contracts:migrate
```

### Test Contracts

```
yarn contracts:test
```
